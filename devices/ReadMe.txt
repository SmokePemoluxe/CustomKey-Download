Шаблон Sample500x300.psd
Предназначен для создания собственных значков клавиш в ПО CustomKey при помощи графических редакторов.
Ширина холста в 500 пикселей не должна изменяться. Высота должна быть кратна 100 пикселям.
Каждые 100х100 пикселей - это один значок для клавиши. Каждая последующая ячейка соответствует следующей кнопке, начиная отсчет слева направо.
